# Stacks and Queues

"Linear Data Structures"

## Stacks

"LIFO" - Last In First Out, think of them as a stack of plates
- Pop: O(1);
- Push O(1);
- Peek O(1);
- Lookup: O(n);


## Queues 

"FIFO" - FIFO, First In First Out, think of an entrance to a roller coaster 
- Enqueue: O(1);
- Dequeue: O(1);
- Peek: O(1);
- Lookup: O(n);
